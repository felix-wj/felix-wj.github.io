              # scoop

              使用指南 (opens new window)

              将 Scoop 安装到自定义目录 :

              $env:SCOOP='D:\Scoop'
# 先添加用户级别的环境变量 SCOOP
[environment]::setEnvironmentVariable('SCOOP',$env:SCOOP,'User')
# 然后下载安装 Scoop （如果使用默认安装路径则直接运行下面的命令）
iex (new-object net.webclient).downloadstring('https://get.scoop.sh')
# 或者使用下面的命令安装：
iwr -useb get.scoop.sh | iex

#设置代理
scoop config proxy 127.0.0.1:7890
#取消代理设置
scoop config rm proxy

              # winget

              1. 搜索软件

              winget search notepad++

              1. 安装软件

              使用-l指定安装路径安装app winget install notepad++ -l d:\notepad++

              1. 列出已安装应用

              1. 升级应用

              1. 卸载应用

              winget uninstall

              # Terminal

              # Git Bash

              打开 Windows Terminal，点击顶部下拉箭头 ➔ 设置（或按 Ctrl + ,）,点击底部的打开JSON文件

              添加 Git Bash 配置 ,参数如下：

              • commandline: Git Bash 可执行文件路径（-i -l 表示启动交互式登录 Shell）。

              • name: 终端中显示的名称。

              • icon: Git Bash 图标路径（可选）。

              • startingDirectory: 启动时的默认目录（%USERPROFILE% 为用户文件夹）。

              设为默认终端（可选）

              在 settings.json 的根节点添加以下配置，将 Git Bash 设为默认启动项：

              "defaultProfile": "{Git Bash 配置的 GUID}"
              1

              示例配置：

              {
  "profiles": {
    "list": [
      {
        "guid": "{00000000-0000-0000-0000-000000000001}",
        "commandline": "\"%PROGRAMFILES%\\Git\\bin\\bash.exe\" -i -l",
        "name": "Git Bash",
        "icon": "%PROGRAMFILES%\\Git\\mingw64\\share\\git\\git-for-windows.ico",
        "startingDirectory": "%USERPROFILE%"
      }
    ]
  },
  "defaultProfile": "{00000000-0000-0000-0000-000000000001}"
}

              # 传输文件

              1. 上传本地文件到服务器

              scp /path/filename username@servername:/path/

              1. 从服务器上下载文件

              scp username@servername:/path/filename /var/www/local_dir

              1. 从服务器下载整个目录

              scp -r username@servername:/var/www/remote_dir/ /var/www/local_dir

              1. 上传目录到服务器

              scp -r local_dir username@servername:remote_dir
